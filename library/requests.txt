Требования к библиотеке консольного управления заметками (To-Do)

1. Класс Note (заметка)
   - Содержит поля: описание (строка), статус (bool – выполнено/не выполнено), приоритет (int 1-3, где 1 – высокий, 3 – низкий)
   - Конструктор: принимает описание и приоритет, статус по умолчанию false
   - Методы: getDescription(), getPriority(), isDone(), setDone(bool)

2. Класс Notebook (блокнот)
   - Хранит список заметок
   - Методы:
     * addNote(описание, приоритет) – добавить новую заметку
     * listNotes() – вывести все заметки с номерами, описанием, приоритетом и статусом
     * markDone(номер) – отметить заметку по номеру (1-нумерация) как выполненную
     * deleteNote(номер) – удалить заметку
     * size() – количество заметок

3. Консольное меню (класс Menu или функции)
   - Бесконечный цикл с выводом пунктов:
     1. Добавить заметку
     2. Показать все заметки
     3. Отметить заметку выполненной
     4. Удалить заметку
     5. Выход
   - При выборе пункта запрашивает нужные данные у пользователя
   - Обрабатывает ошибки ввода (не число, число вне диапазона, пустое описание, приоритет не 1-3)
   - Использует валидацию для ввода строки (описание не пустое) и числа (приоритет 1-3)

4. Библиотека предоставляет:
   - Заголовочный файл lib.hpp (объявления классов)
   - Файл lib.cpp (реализация)
   - Функцию runNotebookApp() для запуска приложения
